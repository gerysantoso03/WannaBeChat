## Description

Realtime chat room with express.js and socket.io

## Source

- [Socket IO](https://socket.io/)
- [Moment JS](https://momentjs.com/)

## Instalation

#### 1. Install Dependencies

```
npm install
```

#### 2. Run in Development mode

```
npm run dev
```

#### 3. Run in Production mode

```
npm start
```

#### 4. Open your localhost

```
http:/localhost:3000
```
